Space Shooter

Descripción

Space Shooter es un juego desarrollado en Python utilizando la librería PyGame. El objetivo del juego es controlar una nave espacial para evitar colisiones con asteroides y destruirlos disparando láseres.

Requisitos

Python 3.10 o superior

PyGame instalado (pip install pygame)

Instrucciones de Juego

Ejecuta el archivo main.py para iniciar el juego.

Controla la nave espacial con las teclas de dirección:

Izquierda (←): Mueve la nave a la izquierda.

Derecha (→): Mueve la nave a la derecha.

Espacio: Dispara un láser.

Evita colisiones con los asteroides y dispara para destruirlos.

El juego continúa hasta que cierres la ventana.
